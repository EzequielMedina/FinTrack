FinTrack — Gestión Financiera con Chatbot y Reportes

Proyecto/Tesis — [Tu Nombre]

Presentación enfocada en propuesta de valor, modelo y diferenciadores.

Problema y Oportunidad

  • Datos financieros dispersos (bancos, tarjetas, planillas).
  • Control de cuotas y vencimientos complejo y manual.
  • Reportes poco claros para decisiones rápidas.
  • Oportunidad: automatización y visualización simple para finanzas personales.

Propuesta de Valor

  • Centraliza cuentas, tarjetas y transacciones.
  • Automatiza cuotas, vencimientos y saldos.
  • Reportes y analytics listos para usar.
  • Chatbot para consultas en lenguaje natural.

Producto: Experiencia del Usuario

Dashboard
  • Visión rápida de gastos/ingresos y tendencias.
  • Acceso a reportes PDF/HTML.
Tarjetas y Cuotas
  • Gestión de cuotas, vencimientos y saldos pendientes.
  • Alertas y recordatorios.

Diferenciadores

  • Arquitectura modular en microservicios (escalable y mantenible).
  • Despliegue reproducible con Docker Compose + NGINX.
  • Reportes PDF/HTML, métricas y analytics.
  • Chatbot con persistencia de conversación y contexto.

Modelo de Negocio

  • Freemium: funciones base y límites de reportes/cuotas.
  • Premium: analytics avanzados, alertas inteligentes, exportaciones.
  • Potencial B2B: instituciones educativas y fintech.

Roadmap y Métricas

  • Integraciones bancarias y APIs de cambio USD.
  • Alertas inteligentes y dashboards avanzados.
  • Métricas: usuarios activos, uso del chatbot, reportes generados, tiempo ahorrado.

Demo y Cierre

  • Demo: dashboard, tarjetas/cuotas, chatbot respondiendo preguntas.
  • Beneficios: claridad, automatización y experiencia conversacional.
  • Despliegue: Docker Compose y verificación de logs.

Comandos (PowerShell): docker compose up --build y docker compose logs -f
